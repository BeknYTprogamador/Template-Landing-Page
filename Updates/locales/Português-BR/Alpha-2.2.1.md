# Update Alpha 2.2.1 - Expansão da Documentação

## Propósito
Aprimorar a documentação do projeto consolidando notas de versão e fornecendo arquivos README multilíngues alinhados com os idiomas suportados pela landing page.

## Alterações Implementadas
- Adicionados arquivos README traduzidos na pasta `documentation/`:
  - `README(English).md`
  - `README(PT-BR).md`
  - Futuras versões de README para outros idiomas seguirão a mesma estrutura.
- Organizada a pasta `updates/locales/` para separar notas de versão por idioma.
- Garantida consistência entre os arquivos de documentação para facilitar navegação e colaboração.

## Status Atual
- **Funcionando:** Documentação disponível em inglês e português brasileiro.
- **Planejado:** Adição de novos idiomas conforme a landing page expandir o suporte de localização.

## Notas
Esta versão está marcada como **Alpha 2.2.1**. O foco está na completude da documentação e na acessibilidade multilíngue, em vez de novos recursos. Com esta atualização, a documentação do projeto está alinhada com os objetivos de internacionalização, abrindo caminho para o desenvolvimento da versão Beta.
