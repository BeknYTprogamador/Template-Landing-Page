# Atualização Alpha 2.3 - Suporte ao Espanhol (Espanha)

## Propósito
Expandir a implementação multilíngue adicionando suporte completo para o idioma espanhol (Espanha), consolidando a internacionalização do projeto.

## Alterações Implementadas
- Adicionado o arquivo de tradução `es.json` em `src/locales/` para habilitar conteúdo em espanhol.
- Atualizado o seletor de idioma na barra de navegação para incluir inglês, português brasileiro e espanhol (Espanha).
- Criado `README(ES).md` na pasta `documentation/` para fornecer detalhes completos do projeto em espanhol.
- Garantida a consistência entre os arquivos de tradução (`en.json`, `pt-br.json`, `es.json`) mantendo a mesma estrutura de chaves.

## Status Atual
- **Funcionando:** Todas as seções (Navbar, Hero, Slideshow, Serviços, Sobre, Portfólio, Rodapé) agora se traduzem corretamente entre inglês, português brasileiro e espanhol.
- **Estável:** A base multilíngue está consolidada, pronta para futuras expansões de idiomas.

## Notas
Esta versão está marcada como **Alpha 2.3**. O foco principal é a expansão do suporte a idiomas, reforçando a internacionalização antes de avançar para a Alpha 3.0, que terá como objetivo a responsividade da barra de navegação.
