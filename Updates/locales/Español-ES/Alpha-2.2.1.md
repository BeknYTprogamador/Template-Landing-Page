# Actualización Alpha 2.2.1 - Expansión de la Documentación

## Propósito
Mejorar la documentación del proyecto consolidando notas de versión y proporcionando archivos README multilingües alineados con los idiomas soportados por la página de aterrizaje.

## Cambios implementados
- Se añadieron archivos README traducidos en la carpeta `documentation/`:
  - `README(English).md`
  - `README(PT-BR).md`
  - Las futuras versiones de README para otros idiomas seguirán la misma estructura.
- Se organizó la carpeta `updates/locales/` para separar notas de versión por idioma.
- Se garantizó la consistencia entre los archivos de documentación para facilitar la navegación y la colaboración.

## Estado actual
- **Funciona:** Documentación disponible en inglés y portugués brasileño.
- **Planificado:** Adición de nuevos idiomas conforme la página de aterrizaje amplíe el soporte de localización.

## Notas
Esta versión está marcada como **Alpha 2.2.1**. El enfoque está en la completitud de la documentación y en la accesibilidad multilingüe, en lugar de nuevas funcionalidades. Con esta actualización, la documentación del proyecto está alineada con los objetivos de internacionalización, abriendo el camino hacia el desarrollo de la versión Beta.
